
## 卒制アイデア第２弾

### 背景

この卒制アイデアの原点は、英語学習やロンドンでのプログラミング学習など、自分自身が新しい挑戦を続けてきた経験が元になっています。

海外での慣れない生活や仕事と並行した学習環境では、日々の努力がすぐに成果として見えないことも多く、不安や孤独感を感じる場面もありました。


それでもなぜ学習を安定して続けてこられたのかを、今回の卒制をきっかけに見つめ直してみると、
やっぱりゲームのように進捗を見える化したり、定期的に振り返ったりする仕組み、
ゲーミフィケーションのエッセンスを利用して学習をしていたからだと思っています。

例えば、
「頑張ったらあのパン屋でショートブレッドクッキーを買ってアニメを一気見」
「目標を達成したら友達を誘ってパブでビール」
というご褒美をイギリスでは毎回設定しており、効果絶大でした。

今日・今週の努力が自分の楽しみに変わる感覚があることで、辛い学習や孤独に振り回されずに
学習を継続できたからこそ、イギリスで2年間も生き抜くことができたのだと思っています。


前回の卒制アイデアを紹介した記事でも書きましたが、


自分が本格的に勉強という行為を行い始めたのは7年前で、当時はAIが今のように発達していなかったので、学習目標はExcelやNotion、ご褒美リストはTodoやメモアプリのように、

複数のツールを行き来して管理していました。
その結果、ご褒美の仕組み自体を続けるのが面倒になり、せっかくのモチベーション設計が機能しないことは何度もありました。

### 課題
また根本的な部分に関しては先日、友人にこの卒制ネタを相談した時に話題に上がった
「そもそもなんで学習サボるのか？」という問いに対して、

私たちが出した結論が

「怠惰」

というシンプルな答えでした。

この怠惰によって以下の事象が発生することで負の循環が起きているのだと考えました。

「やることを決めたのにダラダラ」「明日やろう」「決めたことができずに自己嫌悪」という無限ループ

また、別の友人の体験談も思い出しました。

学習管理アプリって、目標管理・グラフ・記録機能が充実しているけど
継続が苦手な自分にとっては機能が多く、使い始めることや使いこなすこと自体が負担。

数字やグラフを見てもモチベーションにつながりにくく、結局「管理すること」が面倒。

などといった話を思い出しました。

私自身もたまに陥るのですが、本当に解決したいのは、学習計画を細かく管理することではなく、
学習という負荷の高い行動に取り組むための短期的な動機づけをつくることだと考えています。

特に、長期的な目標だけで動ける人ではなく、目の前の行動に対して「これをやったら何が得られるのか」が見えたほうが動きやすい人に向けた設計が重要なのではないかと考えています。

### このサービスの焦点、価値

このサービスは、日々の学習時間や努力を「ゴールド」として可視化し、自分で設定したご褒美と交換できるようにすることで、学習を続けるきっかけをつくるアプリです。

長期的な学習計画を完璧に管理することではなく、もっと短期的に、「今日の学習が自分の生活の楽しみに近づいている」と感じられる体験をつくることに焦点を当てています。

学習した時間がポイントとして貯まり、コーヒー、外食、映画、ゲーム、欲しいものなど、自分の生活にリンクした報酬と交換できることで、努力の意味を実感しやすくします。

このサービスの価値は、学習を「我慢して続けるもの」から「頑張った分だけ自分を労われるもの」に変えることです。成果が見えづらい学習でも、短い時間の積み重ねがゴールドとして見えることで、小さな達成感を得られます。また、ご褒美を同じアプリ内で管理できるため、複数ツールを行き来する面倒さを減らせます。
